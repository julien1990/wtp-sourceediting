345         * @param method the {@link IFunction} to be renamed.
346         * @param newName the method's new name. <code>null</code> is a valid value
347         * indicating that no new name is provided.
348         * @param flags flags controlling additional parameters. Valid flags are

354        public static RenameSupport create(IFunction method, String newName, int flags) throws CoreException {
355                JavaRenameProcessor processor;
356                if (MethodChecks.isVirtual(method)) {
357                        processor= new RenameVirtualMethodProcessor(method);
358                } else {
359                        processor= new RenameNonVirtualMethodProcessor(method);
360                }
361                return new RenameSupport(processor, newName, flags);
362        }
