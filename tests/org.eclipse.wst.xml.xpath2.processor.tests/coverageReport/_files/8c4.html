28 * and the type name is generalized to a type so that parameterized

30 * <pre>

32 *        [ Expression <b>.</b> ]

34 *            Type <b>(</b> [ Expression { <b>,</b> Expression } ] <b>)</b>

40 * types cannot be instantiated, and array creations must be represented with
41 * <code>ArrayCreation</code> nodes). The normal use is when the type is a

44 * <p>

46 * <ol>

50 * <li>
51 * <code>SimpleType(QualifiedName(SimpleName("A"),SimpleName("B")))</code>
52 * </li>
53 * </ol>
