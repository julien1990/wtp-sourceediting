25 * A search participant describes a particular extension to a generic search

28 * <p>
29 * A search participant is involved in the indexing phase and in the search phase.
30 * The indexing phase consists in taking one or more search documents, parse them, and
31 * add index entries in an index chosen by the participant. An index is identified by a
32 * path on disk.
33 * The search phase consists in selecting the indexes corresponding to a search pattern
34 * and a search scope, from these indexes the search infrastructure extracts the document paths
35 * that match the search pattern asking the search participant for the corresponding document,
36 * finally the search participant is asked to locate the matches precisely in these search documents.
37 * </p>
