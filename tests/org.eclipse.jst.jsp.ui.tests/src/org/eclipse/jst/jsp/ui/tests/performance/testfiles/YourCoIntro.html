YourCo Sample

See the samples gallery come together on the YourCo intranet web site. As an employee of this fictitious company, you can:

  • Look up employee phone numbers
  • Reserve a conference room
  • View or post job listings
  • Manage your leave accounts
  • Personalize the Employee Center

This sample requires a database to store the YourCo data. If you have not already installed and configured your database for the Web Tooling samples, refer to Database Configuration for instructions.

Run this sample

 

How this sample works:

The YourCo web site has:

  • A login feature that determines if you are an employee of YourCo and then puts your personal data into the session
  • A personalized Employee Center where you can add your own preferences
  • Access to YourCo databases that lets you query, insert, update, and delete data depending on your authority
  • Result pages that contain filtered content based on your profile data
  • A logout feature that removes your personal data from the session

  • The Quote, Expiring HTML, Poll, and Survey samples - adapted for use on this site

Follow these links to find out how these features work and to try them out yourself.

How to use this sample on your web sites:

This entire sample was built using Eclipse WebTools and its companion products. You can reuse the servlets and their pages, modifying them for your company's Web sites. Or, you can build your own using the Eclipse wizards.
