DTD Preference Pages

Steps:

  1. Select Window->Preferences... from the menu bar.
  2. Select General->Web and XML->DTD Files in the left pane.
  3. Verify the translations for this page.
  4. Select General->Web and XML->DTD Files->Syntax Coloring in the left pane.
  5. Verify the translations for this page.

    Note: the Color dialog invoked from the Syntax Coloring preferences pages is from the native operating system. Depending on the locale set in the operating system, the Color dialog may or may not be translated.
  6. Select General->Web and XML->DTD Files->Templates in the left pane.
  7. Verify only the contents in the table (names, contexts, descriptions) *There is currently nothing in the table, so nothing to verify.
  8. Press the Cancel button from the Preferences dialog. Discard any changes.
