New Javascript File Wizard

Steps:

  1. Go to File ->New ->Other ->Web and select JavaScript
  2. Verify description of JavaScript wizard, Create a new JavaScript file.
  3. Click the Next button
  4. Verify the following in the New JavaScript File dialog:
    • New JavaScript Page
    • JavaScript
    • Create a new JavaScript file
    Note: Other translations are covered by IES.
  5. Begin typing in a file name
  6. Verify the error message that appears.
  7. Press Cancel to leave the wizard.
