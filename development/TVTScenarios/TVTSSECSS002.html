New CSS File Wizard

Steps:

  1. Go to File ->New ->Other ->Web and select CSS
  2. Verify description of CSS wizard, Create a new Cascading Style Sheet.
  3. Click the Next button
  4. Verify the following in the New Cascading Style Sheet dialog:
    • New Cascading Style Sheet
    • Cascading Style Sheet
    • Create a new CSS file.
    Note: Other translations are covered by IES.
  5. Type "mycss.cs"
  6. Verify the error message that appears.
  7. Add the last 's' for "mycss.css" then press the Next button.
  8. Verify everything on the Select CSS Template dialog except the Back, Next, Finish, Cancel buttons (those are covered by IES).
  9. Press Finish to create the file and leave the wizard.
  10. Try to create the same exact css file again, "mycss.css" in the same location.
  11. Verify the error message that appears.
