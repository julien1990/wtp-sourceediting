New XML File Wizard

Prerequisites:

  1. File ->New ->Example... ->Examples ->Editing and validating XML files
  2. Press the Finish button

Steps:

  1. Go to File ->New ->Other ->XML and select XML
  2. Verify description of XML wizard, Create a new XML file.
  3. Click the Next button
  4. Verify the following in the XML File dialog:
    • New XML File
    • XML File
    • Create a new XML file.
    Note: Other translations are covered by IES.
  5. Type "myxml.xm"
  6. Verify the error message that appears.
  7. Add the last 'l' for "myxml.xml" then press the Next button.
  8. Verify everything on the Create XML File From dialog
  9. Select Create XML file from a DTD file and press the Next button
  10. Verify the Select DTD File dialog.
  11. Click on a dtd file, then press the Next button.
  12. Verify the Root Element dialog.
  13. Press the Back button two times. Then select Create XML file from a schema file and press Next.
  14. Verify everything on the XML Schema File dialog.
  15. Select a schema file, then press the Next button.
  16. Verify everything on the Select Root Element dialog.
  17. Try to create the same exact xml file again, "myxml.xml" in the same location.
  18. Verify the error message that appears.
