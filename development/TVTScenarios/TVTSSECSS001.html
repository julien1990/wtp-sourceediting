CSS Source Page Editor

Prerequisites:

  1. Open the J2EE Perspective if it is not already open.(Window ->Open Perspective -> J2EE)
  2. Create a new dynamic web project if it does not already exist. (File ->New ->Dynamic Web Project)
  3. Create and open a CSS file if it does not already exist (File ->New ->Other ->Web ->CSS)

Steps:

  1. Position the mouse somewhere in the content area of the Source page and press mouse button 2.
  2. Verify the following items in the context menu that appears (including cascading menus), which is shown below:
    • Undo/Undo Text Change
    • Cut
    • Paste
    • Cleanup Document...
    • Format
    • Format Active Elements
    • Add to Snippets...
    • Properties
    Note: Other menu items are covered by IES.
  3. Select Cleanup Document...
  4. Verify the Cleanup dialog that appears.
  5. Press the Cancel button from the Cleanup dialog
  6. Select Edit from the menu bar.
  7. Verify the following menu items (including cascading menus):
    • Undo Text Change
    • Redo Text Change
    • Expand Selection To (and submenu)
    • Content Assist
    Note: Translations for other menu items are covered by IES.
  8. Select Source from the menu bar.
  9. Verify all menu items.
  10. In the Outline view, hover over toolbar items and verify:
    • Sort
    • Collapse All
  11. Click on the Outline view Menu (the triangle button) and verify Link with Editor.
  12. Close the editor, discarding any changes.
