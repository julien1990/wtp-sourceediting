CSS Preference Pages

Steps:

  1. Select Window->Preferences... from the menu bar.
  2. Select General->Web and XML->CSS Files in the left pane.
  3. Verify the translations for this page.
  4. Select General->Web and XML->CSS Files->Source in the left pane.
  5. Verify the translations for this page.
  6. Select General->Web and XML->CSS Files->Syntax Coloring in the left pane.
  7. Verify the translations for this page.

    Note: the Color dialog invoked from the Syntax Coloring preferences pages is from the native operating system. Depending on the locale set in the operating system, the Color dialog may or may not be translated.
  8. Select General->Web and XML->CSS Files->Templates in the left pane.
  9. Verify only the contents in the table (names, contexts, descriptions)
  10. Press the Cancel button from the Preferences dialog. Discard any changes.
