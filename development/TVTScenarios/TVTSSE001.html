SSE Preference Pages

Steps:

  1. Select Window->Preferences... from the menu bar.
  2. Select General->Editors->Structured Text Editor in the left pane.
  3. Verify the translations on the Appearance and Hovers pages.
  4. Verify the translations of the all the preferences pages in the red box below.
  5. Verify the content of each preferences page.
    Note: Translations for WSDL Files, XML Catalog, XML Schema Files are covered by other teams Tools.

    Note: the Color dialog invoked from the Styles Preferences pages is from the native operating system. Depending on the locale set in the operating system, the Color dialog may or may not be translated.
  6. Press the Cancel button from the Preferences dialog. Discard any changes.
