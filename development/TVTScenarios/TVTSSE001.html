SSE Preference Pages

Steps:

  1. Select Window->Preferences... from the menu bar.
  2. Select General->Editors->Structured Text Editor in the left pane.
  3. Verify the translations on the Appearance and Hovers pages.
  4. Select General->Editors->Text Editors->Hyperlinking in the left pane.
  5. Only verify the translations of the circled items.
  6. Select General->Web and XML in the left pane.
  7. Verify the translations for this page.
  8. Select General->Web and XML->Task Tags in the left pane.
  9. Verify the translations for this page.
  10. Press the Cancel button from the Preferences dialog. Discard any changes.
