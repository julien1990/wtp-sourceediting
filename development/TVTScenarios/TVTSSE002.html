SSE Property Pages

Prerequisites:

  1. Open the J2EE Perspective if it is not already open.(Window ->Open Perspective -> J2EE)
  2. Create a new dynamic web project if it does not already exist. (File ->New ->Dynamic Web Project)
  3. Create a CSS file if it does not already exist (File ->New ->Other ->Web ->CSS)
  4. Create a JSPF file if it does not already exist (File ->New ->Other ->Web ->JSP) *Make sure to use *.jspf extension.

Steps:

  1. Press mouse button 2 on the css file in the Project Explorer or Navigator and select Properties.
  2. Verify "Web Content Settings" in the list of properties on the left side of the Properties dialog is translated.
  3. Click on Web Content Settings and verify the Web Content Settings Property page (including the CSS Profile dropdown list)
  4. Press the Cancel button to exit the dialog.
  5. Press mouse button 2 on the jspf file in the Project Explorer or Navigator and select Properties.
  6. Verify "JSP Fragment" in the list of properties on the left side of the Properties dialog is translated.
  7. Click on JSP Fragment and verify the JSP Fragment Property page (items in dropdown lists are not translated-no need to verify)
  8. Verify "Web Content Settings" in the list of properties on the left side of the Properties dialog is translated.
  9. Click on Web Content Settings and verify the Web Content Settings Property page (including the Document Type and CSS Profile dropdown list)
  10. Press the Cancel button to exit the dialog.
  11. Press mouse button 2 on the web project in the Project Explorer or Navigator and select Properties.
  12. Verify "JSP Fragment" in the list of properties on the left side of the Properties dialog is translated.
  13. Click on JSP Fragment and verify the JSP Fragment Property page (items in dropdown lists are not translated-no need to verify)
  14. Verify "Task Tags" in the list of properties on the left side of the Properties dialog is translated.
  15. Click on Task Tags and verify the Task Tags Property page
  16. Verify "Web Content Settings" in the list of properties on the left side of the Properties dialog is translated.
  17. Click on Web Content Settings and verify the Web Content Settings Property page (including the Document Type and CSS Profile dropdown list)
  18. Press the Cancel button to exit the dialog.
