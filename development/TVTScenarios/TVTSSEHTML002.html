New HTML File Wizard

Steps:

  1. Go to File ->New ->Other ->Web and select HTML
  2. Verify description of HTML wizard, Create a new HTML page.
  3. Click the Next button
  4. Verify the following in the New HTML Page dialog:
    • New HTML Page
    • HTML Page
    • Create a new HTML Page
    Note: Other translations are covered by IES.
  5. Type "myhtml.ht"
  6. Verify the error message that appears.
  7. Add the last 'ml' for "myhtml.html" then press the Next button.
  8. Verify everything on the Select HTML Template dialog except the Back, Next, Finish, Cancel buttons (those are covered by IES).
  9. Press Finish to create the file and leave the wizard.
  10. Try to create the same exact html file again, "myhtml.html" in the same location.
  11. Verify the error message that appears.
