New JSP File Wizard

Steps:

  1. Go to File ->New ->Other ->Web and select JSP
  2. Verify description of JSP wizard, Create a new JavaServer page.
  3. Click the Next button
  4. Verify the following in the New JavaServer Page dialog:
    • New JavaServer Page
    • JavaServer Page
    • Create a new JavaServer Page
    Note: Other translations are covered by IES.
  5. Type "myjsp.js"
  6. Verify the error message that appears.
  7. Add the last 'p' for "myjsp.jsp" then press the Next button.
  8. Verify everything on the Select JSP Template dialog except the Back, Next, Finish, Cancel buttons (those are covered by IES).
  9. Press Finish to create the file and leave the wizard.
  10. Try to create the same exact jsp file again, "myjsp.jsp" in the same location.
  11. Verify the error message that appears.
