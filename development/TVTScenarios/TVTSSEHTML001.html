HTML Source Page Editor

Prerequisites:

  1. Open the J2EE Perspective if it is not already open.(Window ->Open Perspective -> J2EE)
  2. Create a new dynamic web project if it does not already exist. (File ->New ->Dynamic Web Project)
  3. Create and open a HTML file if it does not already exist (File ->New ->Other ->Web ->HTML)

Steps:

  1. Position the mouse over tags (e.g., <HTML>, <HEAD>, and <BODY>) or attributes (type and border) within the content area of the editor
  2. Verify the Hover Helps are translated.
  3. Position the mouse somewhere in the content area of the Source page and press mouse button 2.
  4. Verify the following items in the context menu that appears (including cascading menus), which is shown below:
    • Undo/Undo Text Change
    • Cut
    • Paste
    • Cleanup Document...
    • Format
    • Format Active Elements
    • Add to Snippets...
    • Properties
    Note: Other menu items are covered by IES.
  5. Select Cleanup Document...
  6. Verify the Cleanup dialog that appears.
  7. Press the Cancel button from the Cleanup dialog
  8. Select Edit from the menu bar.
  9. Verify the following menu items (including cascading menus):
    • Undo Text Change
    • Redo Text Change
    • Expand Selection To (and submenu)
    • Show Tooltip Description
    • Content Assist
    • Quick Fix
    Note: Translations for other menu items are covered by IES.
  10. Select Source from the menu bar.
  11. Verify all menu items.
  12. Select Navigate from the menu bar.
  13. Verify Open Selection menu item (Translations for other menu items are covered by IES).
  14. Remove the </html> tag at the end of the file.
  15. Select Edit->Content Assist from the menu bar.
  16. Verify End with "</html" is translated.
  17. In the Outline view, expand all nodes and selecting each one, verify context menu. Some of the items include:
    • Remove
    • Add Before
    • Add After
    • Edit Doctype...
    • Add Child
    • Add Attribute
  18. Also verify any dialogs that pop up when you select menu items with "..."
  19. In the Outline view, hover over toolbar items and verify Collapse All.
  20. Click on the Outline view Menu (the triangle button) and verify Link with Editor.
  21. Close the editor, discarding any changes.
