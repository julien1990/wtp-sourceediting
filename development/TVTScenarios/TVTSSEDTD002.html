New DTD File Wizard

Steps:

  1. Go to File ->New ->Other ->XML and select DTD
  2. Verify description of DTD wizard, Create a new DTD file
  3. Click the Next button
  4. Verify the following in the Create DTD dialog:
    • Create DTD
    • Create a new DTD file.
    • Create a new DTD file. Specify..
    Note: Other translations are covered by IES.
  5. Begin typing in a file name
  6. Verify the error message that appears.
  7. End the file name with .dtd then press the Next button.
  8. Verify everything on the Select DTD Template dialog except the Back, Next, Finish, Cancel buttons (those are covered by IES).
  9. Press Cancel to leave the wizard.
