Snippets View

Prerequisites:

  1. Open the J2EE Perspective if it is not already open.(Window ->Open Perspective -> J2EE)
  2. Create a new J2EE web project if it does not already exist. (File ->New ->J2EE Web Project)
  3. Create and open a JSP file if it does not already exist (File ->New ->Other ->Web ->JSP)

Steps:

  1. Window->Show View->Other...->Basic->Snippets
  2. Verify the translation of Snippets view name. And the JSP snippets category and all its items.
  3. Move the mouse over the toolbar button.
  4. Verify the hover text that appears: Insert...
  5. Click on the View Menu (the triangle button).
  6. Verify the menu items:
    • Insert...
    • Cut
    • Copy
    • Paste
  7. Press the right mouse button in the empty area below the drawers to bring up the context menu. Select Customize...
  8. Verify the following translations in the Customize Palette dialog:
    • New Category
    • Import
    • Export
    • Show/Hide Drawer
    • Always show
    • Always hide
    • Custom:
    • Browse...
    Note: All other translations are covered by other GEF.
  9. Select New->New Category. Then Select New->New Item.
  10. Verify New Item is translated as well as all translations in the right pane except Name:, Description:, and Hide (which are covered by GEF).
  11. Select Unnamed Category in the left pane. Select the Custom: radio button in the right pane. Then press the Browse... button.
  12. Verify the title and the instruction in the dialog. Also verify the following items in the list:
    • CSS
    • DTD
    • HTML
    • JavaScript
    • JSP
    • JSP Fragment
    • JSP Tag Definition
    • XML (Lax)
    (all other items are contributed by other components)
  13. Press the Cancel button to close the Content Type Selection dialog.
  14. Press the Cancel button to close the Customize Palette dialog.
  15. Drag and drop taglib directive snippet item to open JSP editor.
  16. Verify the Insert Template dialog that pops up.
  17. Highlight some text in JSP editor and right-click to bring up context menu
  18. Verify Add to Snippets... context menu item.
  19. Verify New Category dialog.
