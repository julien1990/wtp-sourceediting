Javascript Editor

Prerequisites:

  1. Open the J2EE Perspective if it is not already open.(Window ->Open Perspective -> J2EE)
  2. Create a new J2EE web project if it does not already exist. (File ->New ->J2EE Web Project)
  3. Create and open a JavaScript file if it does not already exist (File ->New ->Other ->Web ->Javascript)

Steps:

  1. Verify the translation of the labels for the 2 tabs (Source, Preview) at the bottom of the editor.
  2. Select the Source tab.
  3. Position the mouse anywhere in the gray margin area on the left and press mouse button 2.
  4. Verify the translation of:
    • Toggle Breakpoints
    • Enable Breakpoints
    • Breakpoint Properties...
    Note: Translations other menu items are covered by IES.
  5. Position the mouse somewhere in the content area of the Source page and press mouse button 2.
  6. Verify the following items in the context menu that appears (including cascading menus), which is shown below:
    • Undo/Undo Text Change
    • Cut
    • Paste
    • Add to Snippets...
    Note: Other menu items are covered by IES.
  7. Select Edit from the menu bar.
  8. Verify the Content assist menu item.
    Note: Translations for other menu items are covered by IES.
  9. Select Source from the menu bar.
  10. Verify all menu items.
  11. Select the Preview tab at the bottom of the JavaScript editor.
  12. Verify the translations on Preview tab
    Note: this particular JavaScript file should have nothing to be previewed.
  13. In the Outline view, hover over toolbar items and verify:
    • Sort
    • Collapse All
  14. Click on the Outline view Menu (the triangle button).
  15. Verify the menu items:
    • Link with Editor
    • Show Hierarchy
    • Show Variables
  16. Close the editor, discarding any changes.
