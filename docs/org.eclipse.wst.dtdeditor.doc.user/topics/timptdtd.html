Importing DTDs

If you want to work with DTD files that you created outside of the product, you can import them into the workbench and open them in the DTD editor.

To import a DTD into the workbench, follow these steps:

  1. Click File > Import.
  2. Select the import source and click Next.
  3. Fill in the fields in the Import wizard as necessary.
  4. When you are finished, click Finish.
  5. Select the DTD file in the Navigator view and double-click it to open it in the DTD editor.

After you have imported a DTD file into the workbench and opened it in the DTD editor, you can edit it manually.
