Package org.eclipse.wst.xsl.debugger

org.eclipse.wst.xsl.debugger
Class DebugRunner


public class DebugRunner
extends java.lang.Object
The entry point to the debug process which is responsible for configuring a debugger and then communicating with the Eclipse process via sockets using a common set of commands.
  • instantiates an implementation of IXSLDebugger
  • configures the debugger with the transformation pipeline
  • starts the debugger in a separate thread
  • the main thread is then used to listen to incoming requests and call the appropriate debugger methods
