Working in the References Page

In the References page of the Web deployment descriptor editor, you can add or remove references to:
  • an EJB (Enterprise Java™ Bean)
  • a Web service
  • an external resource
  • a message destination
  • an administered object

You can also use this page to specify binding information, properties, and extensions for the WebSphere Application Server

For more detailed information about adding and removing references, see Defining References in J2EE Modules.
