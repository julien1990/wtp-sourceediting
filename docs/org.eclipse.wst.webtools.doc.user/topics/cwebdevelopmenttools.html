Web development tools

The Web development tools enable you to create Web applications that range from simple static sites to dynamic Web applications that use the latest Web technologies.

The editors include standard source editors with content assist features. To extend these pages, you can include form elements, Java™ applets, embedded scripts, dynamic controls, and JavaServer Pages (JSP) tags.
