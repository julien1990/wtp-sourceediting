Creating a WML or cHTML file

To create a new WML or CHTML file, complete the following steps:

  1. From the Web perspective, select File > New > Other > Web > WML File.
  2. Click Next to open the wizard.

    Note: you can also start this wizard by starting the New HTML File Wizard and selecting WML from the Markup Language pull-down.

  3. Click Browse to select the folder in which you want to create the file.

    Make sure you choose a folder under the WebContent folder of the Web project. This ensures that the file is available for validation, publishing, and deployment.

  4. Type a name for your new file.
  5. Ensure that either the WML 1.3 or Compact HTML option is displayed in the Markup Language drop-down list.
  6. You have two options for proceeding:
    • If you want to accept the defaults associated with a new WML or cHTML file, select Finish. By default, you will generate a blank WML or cHTML page.
    • If you want to specify or customize the file options, select the Configure advanced options check box and click Next.
  7. Select the appropriate encoding value to add to the file's WML or cHTML declaration from the Encoding list button. WML files use HTML settings. The default is the value set in the HTML Files preferences page, which is accessed by selecting Window > Preferences, and then expanding the Web and XML Files preferences.
  8. Ensure that the proper content type is selected in the Content Type drop-down list. The default for a WML file is text/vnd.wap.wml; the default for a cHTML file is text/html.
  9. Select the Document Type that you wish to associate with the WML or cHTML file.

    The default for a WML file is WML 1.3; the default for a cHTML file is Compact HTML 1.0 Draft

    .
  10. Click Finish to create the file in the project that you have selected.
