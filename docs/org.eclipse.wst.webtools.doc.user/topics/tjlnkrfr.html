Refreshing the Links view

To view updates in the Links view:

  1. Ensure that you have saved changes to all of the files you want to view links between.
  2. Select the Links tab or open the Links view.
  3. In the Project Explorer view, click on the file you want to have center focus in the Links view.

Updates, such as changes of link attributes, renaming of files and new links are displayed in the Links view.
