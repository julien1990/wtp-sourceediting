Setting the XML source suggestion strategy used by content assist

You can customize the formatting of the suggestions given by content assist by choosing a suggestion strategy preference. When the suggestion strategy is set to "Strict", suggestions that are 'strictly' grammatically valid will be displayed first in the list of options and highlighted by bold icons. Other suggestions that are applicable to this element's scope but are not grammatically valid, will be displayed lower in the list with de-emphasized icons. When the suggestion strategy is set to "Lax", all suggestions will displayed in the list with de-emphasized icons.

To set the suggestion strategy used by content assist:
  1. From the menu bar, Click Window > Preferences
  2. Select Web and XML > XML Files > XML Source.
  3. Beside Suggestion strategy in the Content assist section, select the appropriate setting (for example, Strict.
