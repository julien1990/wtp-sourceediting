Specifying XML default encoding and line delimiters

This page lets you specify the line delimiter and the text encoding that will be used when you create or save an XML related file.

To specify the line delimiter you want to use:

  1. Click Window > Preferences > Web and XML files > XML Files.
  2. From the Line delimiter field, select the operating system that applies to your development or deployment platform.
  3. From the Encoding field, specify the default character encoding set that is used when creating XML related files (DTDs, XML files, XML schemas). Changing the encoding causes any new, XML related files that are created from scratch to use the selected encoding.
  4. Click Apply, then OK.
