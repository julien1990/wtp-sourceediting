Specifying line delimiters

Various development platforms use different line delimiters to indicate the start of a new line. Select the operating system that applies to your development or deployment platform.

The following instructions were written for the XML perspective, but they will also work in many other perspectives.

To specify the line delimiter you want to use:

  1. Click Window > Preferences > Web and XML files > XML Files.
  2. From the Line delimiter field, select the operating system that applies to your development or deployment platform.
  3. Click Apply, then OK.
