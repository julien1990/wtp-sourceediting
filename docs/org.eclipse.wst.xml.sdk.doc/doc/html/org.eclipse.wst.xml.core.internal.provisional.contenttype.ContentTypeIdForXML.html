Class ContentTypeIdForXML


public class ContentTypeIdForXML
extends java.lang.Object
This class, with its few field, is a convience to provide compile-time safety when refering to a contentType ID. The value of the contenttype id field must match what is specified in plugin.xml file.

Field Summary

     This value is public only for testing and special infrastructure. The constant nor is value should not be referenced by clients. The value of the contenttype id field must match what is specified in plugin.xml file. Note: this value is intentially set with default protected method so it will not be inlined.
public static java.lang.StringContentTypeID_XML
     This content type is actually supplied by base Eclipse. Its given here just as documentation for WTP based clients. Typically, clients should use the values/constants supplied by base Eclipse.
