org.eclipse.wst.xml.core.internal.provisional.document
Class IDOMNode


public class IDOMNode
extends
This interface describes the extended functionality of our source-oriented DOM. First, our nodes extend the w3c Node interface, IndexedRegion, and INodeNotifier. Plus, has the extra methods called out here. ISSUE: the 'read-only' API should be broken out in their own interface
